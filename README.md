# FA Flutter UI Kit

Official FA Flutter UI Kit.

## Getting Started

- Import the 'package:fa_flutter_ui_kit/fa_flutter_ui_kit.dart' to get all the widgets in your projects that are avilable in FA Flutter UI Kit.FieldAssist
